## Capture Tips

1. **Browser**: Use Chrome or Firefox with DevTools
2. **Resolution**: Set viewport to 1920x1080 for full-page shots
3. **Format**: Save as PNG for best quality
4. **Tool**: Use browser DevTools screenshot feature:
   - Chrome: `Cmd+Shift+P` → "Capture full size screenshot"
   - Firefox: `Cmd+Shift+S` → Select area

## Quick Capture Commands

```bash
# macOS
screencapture -i docs/screenshots/homepage-dark.png

# Windows (Snipping Tool)
# Win+Shift+S, then save to this folder
```
